# AI Skill: Confidence-Driven Trade-Off Strategy

## Purpose
Balance **speed** versus **quality** of output based on confidence in problem understanding and solution validity.

## When to Use
- Product development
- Heuristic decision cycles

## Skill Description
Evaluate confidence axes:
- Confidence in problem importance
- Confidence in solution correctness

Decision bias:
- **Low confidence in importance** → prioritize speed
- **High confidence both** → prioritize quality
- **Mixed confidence** → balanced exploration

## Input
- Confidence estimates
- Context requirements

## Output
- Recommended strategy (speed/quality focus)
- Rationale summary

## Technique
Translate continuous confidence measures into strategic choices.
