# AI Skill: Connection Circles System Mapping

## Purpose
Enable an AI agent to **visualize and reason about complex causal relationships** among system components and identify feedback loops.

## When to Use
- Exploring cause‑and‑effect structures in a system
- Identifying reinforcing or balancing loops
- Hypothesizing interdependencies and leverage points

## Skill Description
This skill arranges system elements around a conceptual circle and connects them via directed edges (causal links) with polarity (+/–) to map influences.

## Input
- Set of relevant system variables (concepts/entities)
- Observations or hypotheses about causal links

## Output
- A connection map of system relationships
- Identified feedback loops (reinforcing or balancing)
- Structural insights about systemic behaviour

## Technique
1. List key variables relevant to the problem.
2. Connect each pair where causality is justified.
3. Annotate arrows with signs: “+” for same‑direction impact, “–” for inverse impact.
4. Trace closed loops to find feedback structures.

Use loops to explain persistence or resistance in system behaviour.
