# AI Skill: Balancing Feedback Loop Analysis

## Purpose
Enable an AI agent to identify **balancing feedback loops** that **stabilize a system** in response to disturbances.

## When to Use
- Understanding system stability
- Regulatory and control problem analysis
- Policy design where equilibrium matters

## Skill Description
This skill recognizes circular causal structures where changes trigger self‑correcting adjustments that reduce divergence from a desired state.

## Input
- A system description with variables and indicators
- Relevant behavioural data

## Output
- Identification of balancing loops
- Loop explanations
- Insight into system resistance and equilibrium targets

## Technique
1. Identify variables with goal vs. actual mismatch.
2. Trace causal sequences that act to reduce the gap.
3. Confirm odd‑number sign inversion in loop for balancing classification.
4. Summarize stabilizing mechanism and implications.

Distinguish balancing loops from reinforcing ones to understand long‑term trends.
