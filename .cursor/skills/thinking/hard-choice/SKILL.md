# AI Skill: Hard Choice Classification

## Purpose
Determine the **type of decision** and guide the nature of the solution strategy.

## When to Use
- Complex decision governance
- Meta-reasoning about choice difficulty

## Skill Description
Assess decisions along:
- Impact (low/high)
- Comparability (easy/hard to compare)

Decision types:
- **No-brainer**
- **Apples & Oranges**
- **Big choice**
- **Hard choice**

## Input
- Candidate options
- Comparability signal
- Impact estimate

## Output
- Decision type classification
- Strategic guidance for next steps

## Technique
Use classification to adapt reasoning depth and resource allocation.
