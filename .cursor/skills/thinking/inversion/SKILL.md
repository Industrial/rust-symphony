# AI Skill: Inversion Reasoning

## Purpose
Empower the agent to approach problems from **reverse or opposite perspectives** to uncover pitfalls, blind spots, and robust solutions.

## When to Use
- When the problem seems constrained by assumptions
- Risk mitigation and failure mode analysis
- Pre‑mortem and stress testing

## Skill Description
This skill prompts:
1. **Worst‑case enumeration** (“What would be the worst choice?”)
2. **Failure cause analysis**
3. **Reformulation toward robust alternatives**

## Input
- Proposed solution or goal
- Known assumptions

## Output
- List of failure scenarios
- Adjusted solution directions that avoid pitfalls

## Technique
Invert the goal, enumerate failure causes, then convert insights into criteria that must be avoided in forward directions.
