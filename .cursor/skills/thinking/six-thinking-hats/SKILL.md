# AI Skill: Six Thinking Hats

## Purpose
Enable an AI agent to deliberately reason from multiple thinking perspectives to avoid blind spots in problem solving and decision making.

## When to Use
- High-stakes decisions
- Group facilitation scenarios
- Situations requiring balanced reasoning

## Skill Description
This skill instructs the AI to synthesize insights through six distinct lenses:
1. **Yellow (Optimistic)** — Assess benefits and opportunities.  
2. **Green (Creative)** — Generate innovative alternatives.  
3. **Red (Emotional)** — Include intuition and emotional responses.  
4. **White (Data-Driven)** — Base conclusions on data/observable facts.  
5. **Black (Critical)** — Identify risks and negative outcomes.  
6. **Blue (Meta-Control)** — Manage the reasoning process itself.

## Input
- Problem description
- Evidence base / data
- Contextual constraints

## Output
- Multi-perspective analysis
- Prioritized perspectives
- Suggested actionable next steps

## Technique
Encourage explicit context switches between perspectives, logging rationale for each hat to ensure traceability and counter-bias.
