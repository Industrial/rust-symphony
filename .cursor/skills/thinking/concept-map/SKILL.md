# AI Skill: Concept Mapping

## Purpose
Equip an AI agent with the ability to **create structured representations of knowledge**, showing how concepts connect and interact within a domain.

## When to Use
- Knowledge organization and explanation
- Domain exploration and learning
- Clarifying concept interrelations

## Skill Description
This skill produces a **graph of nodes (concepts)** and **links (relationships)** where each link is labelled with a descriptive verb phrase that explains how the concepts relate.

## Input
- A focus question defining scope
- A list of relevant concepts/entities

## Output
- A concept map: nodes + labeled relationships
- Insight into structural knowledge gaps

## Technique
1. Formulate a focus question guiding the mapping.
2. Identify and prioritize key concepts/entities.
3. Place concepts visually and connect them with labeled relationships.
4. Iterate to fill gaps, refine links, or adjust hierarchy.

Concept maps can be saved as structured graph data (nodes + edges) for further reasoning or querying.
