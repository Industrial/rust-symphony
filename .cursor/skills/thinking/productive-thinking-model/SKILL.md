# AI Skill: Productive Thinking Workflow

## Purpose
Structure an AI agent’s **problem understanding to solution planning** process using a six‑step productive thinking framework.

## When to Use
- Open‑ended problem situations
- Team collaboration scenarios
- Creative problem solving

## Skill Description
This skill enables an agent to move through a structured pipeline:
1. **Understand** “What’s going on?”
2. **Define success** and desired outcomes
3. **Frame catalytic questions**
4. **Generate answers**
5. **Select and refine solutions**
6. **Align resources for execution**

## Input
- Problem context
- Stakeholder values and constraints

## Output
- Catalytic questions
- Candidate solutions with evaluation
- Execution plan with resource assignments

## Technique
Follow an iterative six‑step workflow to clarify, generate, evaluate, and operationalize solutions. Use decision matrices and other supporting tools during evaluation stages.
