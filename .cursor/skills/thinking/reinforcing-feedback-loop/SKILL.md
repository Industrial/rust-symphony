# AI Skill: Reinforcing Feedback Loop Analysis

## Purpose
Enable an AI agent to detect **reinforcing feedback loops** that **amplify change** within systems, leading to exponential growth or decline.

## When to Use
- Systems exhibiting accelerating or compounding behaviour
- Growth/decay analysis
- Identifying virtuous or vicious cycles

## Skill Description
This skill traces circular causation where the output of one iteration acts as an input that amplifies in the next iteration, generating non‑linear dynamics.

## Input
- System variables and causal assumptions
- Temporal or trend data if available

## Output
- Mapped reinforcing feedback loops
- Explains potential exponential behaviours
- Suggests intervention points to tame or leverage amplification

## Technique
1. Trace causal sequences returning to the starting variable.
2. Confirm even number of negative polarity links (including zero negatives) for reinforcing classification.
3. Document expected amplification trajectory and rate drivers.

Use understanding of reinforcing loops to forecast long‑term system behaviour.
