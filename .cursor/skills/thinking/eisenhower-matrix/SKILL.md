# AI Skill: Eisenhower Matrix Prioritization

## Purpose
Prioritize tasks or actions by categorizing them on **importance** and **urgency** to optimize decision sequencing.

## When to Use
- Work prioritization
- Scheduling
- Task triage

## Skill Description
Classify tasks into four categories:
- **Do Now** — Important & urgent
- **Schedule** — Important but not urgent
- **Delegate** — Urgent but not important
- **Ignore** — Neither important nor urgent

## Input
- List of tasks or candidate actions
- Assigned importance and urgency scores

## Output
- Categorized task list
- Priority ordering
- Recommended handling strategy

## Technique
Quantify urgency and importance; optimize decisions based on resource constraints and strategic impact.
