# AI Skill: Cynefin Situation Classification

## Purpose
Classify problem contexts to select the most appropriate reasoning strategy.

## When to Use
- Ambiguous or poorly understood scenarios
- Large scale system decisions

## Skill Description
Assign situations into Cynefin domains:
- **Clear** — obvious cause/effect
- **Complicated** — analyzable with expertise
- **Complex** — emergent patterns
- **Chaotic** — unstable with no clear patterns
- **Disorder** — unclear domain

Each domain drives a tailored response strategy.

## Input
- Situation description
- Available evidence

## Output
- Domain classification
- Strategy recommendation

## Technique
Contextual profiling to reduce risk of mismatched reasoning types.
