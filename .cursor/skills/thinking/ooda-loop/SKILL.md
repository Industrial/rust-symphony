# AI Skill: OODA Loop Adaptive Reasoning

## Purpose
Rapid iteration through decision cycles under uncertainty.

## When to Use
- Fast changing environments
- Competitive contexts

## Skill Description
Continuously loop through phases:
1. **Observe** current state and environment
2. **Orient** by analyzing and synthesizing
3. **Decide** on a course of action
4. **Act** to test choice

## Input
- Current state data
- Environmental signals
- Feedback from prior cycles

## Output
- Updated action decisions
- Model of environment
- Learning updates

## Technique
Implement tight feedback cycles to enable rapid adaptation.
