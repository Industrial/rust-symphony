# AI Skill: Second-Order Consequence Reasoning

## Purpose
Enable deeper foresight by examining long-term and cascading outcomes, beyond first-order effects.

## When to Use
- Strategic planning
- Risk forecasting
- Policy design

## Skill Description
This skill prompts the model to iteratively ask “**And then what?**” to uncover multi-stage consequences of decisions.

## Input
- Action or strategy under consideration
- Contextual timeline parameters

## Output
- First, second, and optionally third order consequences
- Probability estimates for outcomes where feasible

## Technique
Chain reasoning across extended consequence paths; log assumptions at each step.
