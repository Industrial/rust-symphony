# AI Skill: Impact-Effort Prioritization

## Purpose
Prioritize initiatives based on expected payoff relative to required effort.

## When to Use
- Backlog prioritization
- Resource planning
- Execution sequencing

## Skill Description
Classify tasks into four quadrants:
- **Quick Wins**: high impact, low effort.
- **Major Projects**: high impact, high effort.
- **Fill-ins**: low impact, low effort.
- **Avoid**: low impact, high effort.

## Input
- Candidate tasks
- Estimated effort
- Expected impact

## Output
- Prioritized roster
- Allocation recommendations

## Technique
Use impact/effort axis placements as heuristic decision filters.
