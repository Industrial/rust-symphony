# AI Skill: Ladder of Inference Debugger

## Purpose
Expose and correct hidden assumptions leading to flawed conclusions.

## When to Use
- Bias mitigation
- Explanation generation
- Trustworthy reasoning

## Skill Description
Trace reasoning from raw observations to results, challenging each step:
1. Raw data
2. Selected data
3. Interpretations
4. Assumptions
5. Conclusions
6. Beliefs
7. Actions

## Input
- Inferred conclusion
- Inputs observed

## Output
- Step-by-step reasoning trace
- Flags for unwarranted assumptions
- Revised conclusion where appropriate

## Technique
Hit each ladder rung explicitly and require justifications grounded in evidence.
