# AI Skill: Issue Tree Decomposition

## Purpose
Enable the agent to **decompose complex problems hierarchically** into a systematic tree of subproblems (problem trees) or sub‑solutions (solution trees).

## When to Use
- Complex problem analysis
- Root cause identification
- Scoping and prioritization

## Skill Description
This skill allows an agent to:
- Break down a problem into MECE (Mutually Exclusive, Collectively Exhaustive) branches.
- Construct problem trees (why analysis)
- Construct solution trees (how analysis)

## Input
- High‑level problem statement

## Output
- Hierarchical problem structure
- Candidate subproblems prioritized for investigation

## Technique
1. Identify primary branches of the problem.
2. Grow sub‑branches via rigorous “why/how” analysis.
3. Label branches with hypotheses or solution approaches.

Use MECE principles to avoid overlap and ensure coverage.
