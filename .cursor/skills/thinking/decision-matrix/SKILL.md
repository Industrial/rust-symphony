# AI Skill: Multi-Factor Decision Matrix

## Purpose
Objective evaluation of options across weighted criteria to identify the highest-value choice.

## When to Use
- Complex option selection scenarios
- Product feature prioritization
- Investment decision support

## Skill Description
Build a table of options and factors; assign and weigh each factor; compute aggregate scores to choose the best option.

## Input
- Candidate options
- Decision criteria
- Weights per criterion

## Output
- Option scores
- Ranking
- Recommended selection with explanation

## Technique
Standardize scoring scales and normalize weights to ensure fair comparison.
